## ğŸ§  How It Works

1. **Add Expense**  
   - Enter a title, select category, and input amount  
   - Click **Add** to save entry

2. **Edit Expense**  
   - Right-click on any row to open **context menu**  
   - Select **Edit**, update values, and click **Save**

3. **Delete Expense**  
   - Right-click on any row, select **Delete** to remove the entry

4. **Filter & Sort**  
   - Filter expenses by category using dropdown  
   - Sort amounts ascending/descending via arrows in the table header

5. **Dynamic Total**  
   - Total expense amount updates automatically as you add/edit/delete
