## 📁 Project Structure

Each project has its own folder, usually containing:

```
project-name/
│
├── index.html          # Main HTML file
├── style.css           # Stylesheet
├── images/             # Images or icons used
├── script.js           # JS file (if used)
└── assets/             # Optional assets folder
```
